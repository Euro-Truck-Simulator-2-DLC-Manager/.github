## 🔧 **Troubleshooting**

| Problem                        | Solution                                                                 |
|---------------------------------|--------------------------------------------------------------------------|
| DLCs not showing in-game        | Ensure CreamAPI was run as Administrator and restart Steam.              |
| Game crashes after launch       | Use the “Uninstall” option in CreamAPI and reinstall unlockers.         |
| Missing DLL errors              | Reinstall Koaloader via CreamInstaller’s context menu.                   |
| Still doesn’t work              | Re-run CreamAPI and refresh Steam appinfo for ETS2.    |

## 🧠 **Tips & Notes**

- **Back up your ETS2 folder** before making changes.
- **SteamCMD** will download automatically if missing.
- Works with **official Steam builds only** (no cracked or repack versions).
- If **ETS2** updates, simply rerun **CreamAPI** to reapply the unlockers.
